# Endless Relay (prototype)

This repository contains a Godot 4 prototype for "Endless Relay".

Quick start:
1. Open the project in Godot 4.x.
2. Run `res://scenes/main.tscn`.
3. If input actions are missing, either run the editor script `res://scripts/editor/register_inputs.gd` from the Godot Script Editor or set them manually in Project Settings -> Input Map: move_left, move_right, sprint, use_momentum, jump.

Files added:
- scenes/: scene files
- scripts/: GDScript logic
- ui/: HUD scene and script
- config/: gameplay tunables
